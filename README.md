# Event-Tickets

Display and filter Elton John events!

## Dependencies

- Node.js
- Nodemon
- Express
- React
- Webpack
- Babel

## Getting Started

#### Webpack

If making modifications to `src` files:
`npm run react-dev`

#### Server

To run application:
`npm run server-dev`

Then open the app at
`http://localhost:3000/`
